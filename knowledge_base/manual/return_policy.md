# 退换货政策（节选手册）

> 本文件故意写得偏长，用来对比「整本塞进 Prompt」和「按需 read」。
> 真实企业手册往往几十页——窗口装不下，硬塞只会贵、慢、截断、乱编。

## 1. 适用范围

本政策适用于自营平台标价商品（不含第三方代发、预售特批与虚拟商品）。

## 2. 七天无理由退货

自签收之日起七个自然日内，符合下列条件可申请无理由退货：

1. 商品及包装保持完好，不影响二次销售；
2. 附件、赠品、说明书齐全；
3. 未激活账号绑定、未写入不可撤销数据（数字商品另见专章）。

### 2.1 已拆封是否支持七天无理由？

**一般情况：不支持「七天无理由」。**  
商品外包装或塑封已拆除、撕毁防伪标、试用痕迹明显的，视为影响二次销售，只能走「质量问题退换」（见第 3 章），不能按无理由退货处理。

例外（仍可评估无理由）：

- 包装仅为便于开箱检查的「试用袋」且商品本体未使用；
- 客服书面确认「支持试穿/试用后退货」的活动商品（以订单备注为准）。

回答用户时请引用本小节路径：`manual/return_policy.md` §2.1。

## 3. 质量问题退换

出现功能故障、发错货、物流破损时，不受「已拆封」限制，可申请换货或退货。需提供：

- 订单号、开箱视频或清晰照片；
- 故障现象与发生时间。

审核通过后，运费由平台承担（偏远地区以运费政策为准）。

## 4. 退款时效

- 无理由：仓库验货合格后 3 个工作日内原路退款；
- 质量问题：审核通过后 1–5 个工作日（视支付渠道）。

## 5. 不支持退换清单（摘要）

定制刻字、拆封后的食品/贴身衣物、激活后的软件授权码、时效性票务等，详见活动页小字。

## 6. 示例对话备忘（训练用，勿对用户照读）

客服A：用户说「盒子拆了还能七天无理由吗」→ 先查 §2.1，再给质量问题通道。  
客服B：不要承诺「一定退款」，应说明审核条件。
